## 초저 (초1-2) - FUN high / LOAD low / SCAFFOLD high

- 캐릭터 1명이 끝까지 안내 (일관된 화자). 이름·말투 고정.
- 한 화면 = 한 개념. 글자 크게, 의성어·의태어 활용.
- 따라 하기·손동작·소리 내기 등 몸을 쓰는 미니 활동을 30초마다.
- 노래·운율·반복 구조로 기억 돕기.
- 즉각 칭찬 피드백 ("잘했어요"). 실패해도 안전한 분위기.

## 초고 (초3-6) - FUN high / LOAD low-med / SCAFFOLD med

- 첫 5초 안에 질문 또는 의외의 장면으로 호기심 자극.
- 짧은 호흡(세로 9:16, 60-90초)도, 차시형 슬라이드도 모두 통함.
- 중간 퀴즈로 능동 회상 (passive 시청 끊기).
- 실생활 예시 먼저, 용어는 나중 (show-then-name).
- "왜 그럴까?" 예측 -> 확인 구조. 흔한 오개념 짚어 주기.

## 중 (중1-3) - FUN med / LOAD med / SCAFFOLD med

- 적용·문제풀이 중심. 개념 -> 바로 써먹는 예제.
- 인포그래픽·도식으로 관계 시각화 (단순 나열 X).
- 또래가 흔히 틀리는 지점을 미리 보여 주고 교정.
- 약간의 도전·게임화(점수, 단계)로 동기 유지, 정확성은 양보 X.
- 출처 있는 실제 데이터·사례로 신뢰감.

## 고 (고1+) - FUN med / LOAD med-high / SCAFFOLD low-med

- 밀도 높은 설명 OK, 단 구조(개요->전개->정리)가 분명해야.
- 개념 간 연결·추론을 요구하는 활동 (단순 암기 넘어서).
- 실제 평가·수능/내신 맥락과 연결되면 몰입도 상승.
- 다이어그램·수식 애니메이션으로 추상 개념 구체화 (MANIM 적합).
- 군더더기 없는 톤. 과한 캐릭터·연출은 오히려 신뢰 저하.
